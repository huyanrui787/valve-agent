项目名称：某电厂主蒸汽管道阀门采购

一、投标人资格
投标人须具备 ISO9001 质量管理体系认证及 API 6D 管线阀门认证。
近三年同类业绩不少于 3 项，行业为电力。

二、技术要求
1. 阀门类型：球阀，公称通径不低于 DN200。
2. 公称压力不低于 40 bar（PN40）。
3. 设计温度：工作温度不低于 300℃，介质为蒸汽。
4. 阀体材质：316 不锈钢。
5. 驱动方式：电动。
6. 执行标准须符合 API 600 或 API 6D。

三、废标条款
下列情形之一视为无效投标：未按要求密封；投标保证金未到账；
关键技术参数负偏离且未获招标人书面认可；逾期递交投标文件。

四、时间安排
投标截止时间：2026年7月15日。开标时间同投标截止。
